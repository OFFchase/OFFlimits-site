Setup OFFlimits bot

COMMAND: +setup

This will create the following channels:

  • THE-BOTBOOK-LOGS
  • OPERATIONS
  • RESULTS

    • Channels are required for Scrims commands.

    NOTE:

    Do not change channel names.

Scrimsetup

COMMAND: +scrimsetup

This command used to set scrims channels: Sign-in, Team-slots, ID-pass:

  • +set sign-in #channel

    • To set sign-in channel

  • +set slots #channel

    • To set Team-slots channel

  • +set id-pass #channel

    • To set ID-pass channel

  • +set preview

    • To preview scrim channels
