Results

COMMAND: +results

Aliases:

res

Requierments:

  • Setup

    • Usage:

    • Command can only be used at [results-bot] channel.
    • Make sure the channel is empty from unrelated messages.
    • Send points in the following order:

      • TeamName rank kills rank kills rank kills

    • Each team and it's points in separate message.
    • After writing all teams use [+results / +res].
    • Calculation will appear at ᴏᴘᴇʀᴀᴛɪᴏɴꜱ.
    • Wait for calculation to finish then use to view the results leaderboard.

    • +board

      • to view the results leaderboard.
